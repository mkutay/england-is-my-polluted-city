\section{Libraries used}
\noindent Various APIs and external libraries were used in this project for extended functionality:

- \textbf{Gluon Maps} - Maps Library by Gluon (maintainer of JavaFX) that implements OpenStreetMaps

- \textbf{OSGB by DST} - Library to convert British Grid System (Easting / Northing) to Latitude and Longitude

- \textbf{GeographicLib} - Used for geodesic distance calculation

- \textbf{Postcodes.io} - API used to get location \& address data from the specified longitude and latitude on the map

- \textbf{GSON} - A library by Google to convert between Json and Java Objects

- \textbf{World Air Quality Index API} - An API to get real time Air Quality Index Updates from the World Air Quality Index Project
